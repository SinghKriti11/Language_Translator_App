# Language Translator Web App

A simple and interactive **Language Translator Web Application** built using **HTML, CSS, and JavaScript**.
This app allows users to translate text between multiple languages using an external translation API.

---

## 🚀 Features

*  Translate text between multiple languages
*  Swap languages instantly
*  Copy translated text to clipboard
*  Text-to-speech support for both input and output
*  Clean and responsive UI
*  Real-time translation using API

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the application 
* **CSS3** – Styling and responsiveness 
* **JavaScript (Vanilla JS)** – Core functionality 
* **API Used** – MyMemory Translation API

---


## ⚙️ How It Works

1. Enter text in the input box
2. Select source and target languages
3. Click on **"Translate Text"**
4. View translated output instantly

The app sends a request to the translation API and displays the result dynamically.

---

## 🧠 Key Functionalities

* **Dynamic Language Dropdowns**
  Populated using a predefined list of languages 

* **Language Swap Feature**
  Swap source and target languages with one click

* **Speech Synthesis**
  Listen to input/output using browser's speech API

* **Clipboard Support**
  Easily copy text with one click
